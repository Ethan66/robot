    # README

    # 前言：

    • 虽然Tailwind和windi css这些css框架能帮助我们快速的开发，但是有些样式时捆绑使用的，不适合用css原子类开发。为了更好的快速开发，才有了这个库。
    • 一开始将scss文件进行编译，发现编译后失去了全局变量和函数的作用。假如放到node_modules，打包的时候又不会对node_modules中的文件进行重新编译，所以只能外抛源文件，通过上传到npm，unpkg自动拿到仓库的静态资源，就可以直接下载。

    # 下载地址

    请将源文件放到静态资源下，并把里面的变量变成全局变量

    • scss (opens new window)
    • less (opens new window)

    # 滚动条设置：@include w-scroll-bar(10px, 30px, red) || .w-lines-ellipsis(10px, 30px, red)

    // scss用法
.scroll-bar {
    @include w-scroll-bar();
  }

// scss
// 滚动条设置 $width: 宽度；$radius: 圆角; $thumb: 滑块颜色
@mixin w-scroll-bar($width: 5px, $radius: 20px, $thumb: rgb(204, 204, 204)) {
  overflow-y: scroll;
  &::-webkit-scrollbar {
    width: $width;
  }

  &::-webkit-scrollbar-thumb {
    background-color: $thumb;
    border-radius: $radius;
  }
  &::-webkit-scrollbar-track-piece {
    border-radius: $radius;
    background: rgb(236, 236, 236);
  }
}

// less用法
.scroll-bar {
    .w-scroll-bar();
  }

// less
// 滚动条设置 @width: 宽度；@radius: 圆角; @thumb: 滑块颜色
.w-scroll-bar(@width: 5px, @radius: 20px, @thumb: rgb(204, 204, 204)) {
  overflow-y: scroll;
  &::-webkit-scrollbar {
    width: @width;
  }

  &::-webkit-scrollbar-thumb {
    background-color: @thumb;
    border-radius: @radius;
  }
  &::-webkit-scrollbar-track-piece {
    border-radius: @radius;
    background: rgb(236, 236, 236);
  }
}
