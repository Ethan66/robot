    # CSS

    # Flex布局

    弹性盒子由弹性容器(Flex container)和弹性子元素(Flex item)组成。容器存在两根轴：水平主轴(main axis)和垂直交叉轴(crosss axis)

    # 默认情况

    flex-direction: row; flex-wrap: nowrap;

    1. row: 子元素沿水平主轴由左向右排列
    2. nowrap: 让子元素排一行，假如宽度不够的时候全部子元素缩小。

    I'm Item1;

    I'm Item2;

    I'm Item3; 

    <template>
  <div class="flex-container" style="display: flex; background: #ddd; width: 100px; height: 80px;">
    <p class="flex-item" style="background: red;">I'm Item1; </p>
    <p class="flex-item" style="background: orange;">I'm Item2; </p>
    <p class="flex-item" style="background: pink;">I'm Item3; </p>
  </div>
</template>

    # 容器属性

    1、flex-direction: 改变主轴

    flex-direction: row | row-reverse | column | column-reverse;
// row(默认)沿水平主轴由左向右排列、row-reverse沿水平主轴由右向左排列、column沿垂直主轴右上到下和column-reverse。
    1
    2

    2、flex-wrap: 是否让子元素一行显示

    flex-wrap: nowrap | wrap | wrap-reverse;
// nowrap(默认)不换行，让项目一行展示；wrap：宽度不够换行展示；wrap-reverse：换行且反向展示。
    1
    2

    3、flex-flow：将flex-direction和flex-wrap一起设置

    flex-flow: <flex-direction> || <flex-wrap>;
    1

    4、justify-content: 项目在主轴上的对齐方式（看flex-direction的设置）

    justify-content: flex-start | flex-end | center | space-between | space-around;
// flex-start：在主轴上由左或者上开始排列。flex-end：在主轴上由右或者下开始排列。
// center：在主轴上居中排列。
// space-between：在主轴上左右两端或者上下两端开始排列。
// space-around：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。
    1
    2
    3
    4
    5

    5、align-items: 项目在非主轴上的对齐方式

    align-items: flex-start | flex-end | center | baseline | stretch;
// baseline: 文字在水平一条线上
// stretch: 让项目在非主轴上撑满

    # 项目属性

    1、order: 项目在主轴上的排列顺序。数值越小，排列越靠前，默认为0。

    order: <integer>
    1

    2、flex-grow：项目的放大比例，默认为0，即如果存在剩余空间，也不放大。

    flex-grow: <number>;
    1

    3、flex-shrink：项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。

    flex-shrink: <number>;
    1

    4、flex-basis：项目的初始长度。

    flex-basis: <number> || auto || initial;
    1

    5、flex: flex-grow flex-shrink flex-basis

    # 响应式布局

    # 视口

    viewport（视口）就是浏览器中呈现网页的区域。移动端与 PC 端视口不同，PC端的视口宽度等于分辨率，而移动端的视口宽度跟分辨率没有关系，iOS, Android 基本都将这个视口分辨率设置为 980px，因为是绝大多数PC网页的版心宽度。不过PC网页放到移动端基本会缩放，缩放后文字会变得非常小。后来手机厂商会设置约束视口，宽度大概在320~480 之间。能够保证我们的文字比如 16px，在自己的这个视口下清晰、大小刚刚合适。

    // 解决方案：
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=0" />
width=device-width   视口为设备宽度（就是人设置的一个宽度）//不设置的话默认为980px
initial-scale=1.0    初始化的视口大小是1.0倍
maximum-scale=1.0    最大的倍数是1.0倍
user-scalable=0      不允许缩放视口
// 这个视口的标签告诉浏览器怎么渲染网页。在这里，标签想表达的意思是：按照设备的宽度（device-width）来渲染网页内容。

    # rem响应式布局

    1. 实现原理：rem是根据html的font-size来决定大小的，所以我们用window.resize监听页面宽度来控制html的font-size，从而做到响应式。
    2. 比如我们暂定750px宽度的时候HTML的font-size: 100px，在开发的时候，设计师给的尺寸是375px宽度，我们就给设计师的宽度px除以50，转为rem。在代码里监听页面宽度设置html的font-size。
    3. 高度和某些小图标设定固定值。

    # BFC

    # 什么是BFC

    1. BFC: 块级格式化上下文 (Block Fromatting Context)。他是一个容器，用于管理块级元素，在这个容器中的元素不会影响到其它环境中的布局。
    2. 如何创建BFC：flex元素，浮动元素和绝对定位元素，overflow值不为“visiable”的块级盒子，设置了相对应属性（例如 inline-block、table-cell、table-caption、inline-flex）

    # BFC特性

    1、相邻或嵌套的两个元素（中间没有阻挡：比如没有边框，没有padding）会发生重叠。

    我是marin-bottom: 20px;

    我是margin-top: 20px;

    <template>
  <div style="background: #ddd; overflow: hidden; width: 200px;">
    <p style="margin-bottom: 20px; background:orange;">我是marin-bottom: 20px;</p>
    <p style="margin-top: 20px; background:pink;">我是margin-top: 20px;</p>
  </div>
</template>

    # 易错题

    <div>
  <span>我是span:first-child</span>
  <p>我不是p:first-child</p>

  <p>我不是p:last-child</p>
  <span>我是span:last-child</span>

// p:first-of-type, p:last-of-type表示div的第一个p标签和最后一个p标签

    # 圣杯布局

    1. 给父元素设置左右padding, 位置预留给left，right。这样center可以设置100%，center放最前面，先渲染。三块都设置浮动。
    2. 此时left，right会跳到center的下一行。所以left设置margin-left: 100%; right设置margin-left: -50px; 让left，right回到同一行。
    3. 此时left, right因为padding的关系不是在最左右，所以设置position: relative; 设置left, right将两块移到最左右。

    圣杯布局

    <template>
  <article class="container-sb">
    <div class="center">
      <h2>圣杯布局</h2>
    </div>
    <div class="left"></div>
    <div class="right"></div>
  </article>
</template>
<style>
  .container-sb{
    width: 500px;
    height: 300px;
    padding-left: 100px;
    padding-right: 80px;
    background: #ddd;
  }
  .container-sb .center{
    float: left;
    width: 100%;
    height: 100%;
    background: yellow;
  }
  .container-sb .left{
    float: left;
    width: 100px;
    height: 100%;
    background: red;
    margin-left: -100%;
    position: relative;
    left: -100px;
  }
  .container-sb .right{
    float: left;
    width: 80px;
    height: 100%;
    background: orange;
    margin-left: -80px;
    position: relative;
    right: -50px;
  }
</style>
